# 即梦视频生成本地控制台

本项目提供一个本地 Web UI，用于调用火山引擎即梦视频生成接口。密钥只在本机后端用于签名请求，不会放进浏览器直连第三方。

## 启动

```powershell
python app.py
```

打开：

```text
http://127.0.0.1:7860
```

## 完整使用流程

1. 启动本地服务并打开 `http://127.0.0.1:7860`。
2. 展开左侧“密钥配置”。
3. 手动填写 `Access Key ID` 和 `Secret Access Key`，或导入 Key 文件。
4. 如需长期使用，勾选“保存到本机配置”并点击“保存配置”。未勾选时 Key 只保存在当前页面/本次后端进程中。
5. 在“生成模型”中选择要调用的模型。
6. 填写提示词。
7. 根据模型需要上传首帧图、尾帧图，或填写图片 URL。
8. 设置时长、比例、Seed、运镜模板等参数。
9. 点击“生成视频”。
10. 页面会自动提交任务、轮询任务状态，并在完成后下载 MP4。
11. 生成成功后可在右侧预览、下载，也可以在历史记录里重新打开之前生成的视频。

## 接口调用流程

应用内部使用火山引擎异步视频生成流程：

1. 使用 AK/SK 生成火山引擎请求签名。
2. 调用提交任务接口：

```text
POST https://visual.volcengineapi.com?Action=CVSync2AsyncSubmitTask&Version=2022-08-31
```

3. 获取返回的 `task_id`。
4. 调用查询任务接口轮询状态：

```text
POST https://visual.volcengineapi.com?Action=CVSync2AsyncGetResult&Version=2022-08-31
```

5. 当状态为 `done` 且返回 `video_url` 时，立即下载成本地 `.mp4` 文件。

火山引擎返回的 `video_url` 有时效限制，因此应用会在任务完成后自动下载，避免链接过期。

## Key 文件导入格式

支持 `.txt`、`.env`、`.json`，常见写法都可以：

```text
AccessKeyID: AKLT...
SecretAccessKey: ...
```

```text
VOLC_ACCESSKEY=AKLT...
VOLC_SECRETKEY=...
```

```json
{
  "access_key": "AKLT...",
  "secret_key": "..."
}
```

导入后会先填入页面输入框。勾选“保存到本机配置”并点击“保存配置”后，会写入本机 `.jimeng_config.json`。

## 参数说明

不同模型支持的参数不同，界面会根据模型自动显示或隐藏相关控件。

- `prompt`：提示词，文生视频必填；图生视频也需要配合图片使用。
- `frames`：`121` 表示约 5 秒，`241` 表示约 10 秒。
- `aspect_ratio`：文生视频比例，可选 `16:9`、`9:16`、`1:1`、`4:3`、`3:4`、`21:9`。部分图生视频模型会按输入图片比例处理，不显示该字段。
- `seed`：随机种子，默认 `-1` 表示随机。
- `image_urls`：图片 URL。首帧模型需要 1 张，首尾帧模型需要 2 张。
- `binary_data_base64`：上传本地图片时会自动转成该字段。图片应为 JPEG/PNG，大小和比例限制以官方接口文档为准。
- `template_id`：运镜模型使用的运镜模板。
- `camera_strength`：运镜强度，可选 `weak`、`medium`、`strong`。

## 支持模型

模型列表根据火山引擎即梦视频生成接口文档整理为固定 `req_key` 清单，不是从实时模型列表接口获取。

- 3.0 Pro 1080P：文生视频 / 首帧图生视频
- 3.0 1080P / 720P 文生视频
- 3.0 1080P / 720P 首帧图生视频
- 3.0 1080P / 720P 首尾帧图生视频
- 3.0 720P 运镜图生视频

## 输出位置

生成文件默认保存到：

```text
outputs/
```

每个视频会保存为 `.mp4`，并配套保存一个同名 `.json` 元数据文件，记录任务 ID、请求体和接口返回信息。

## 安全说明

- 不建议把 AK/SK 写入代码或提交到 Git。
- `.jimeng_config.json`、`outputs/`、缓存目录已加入 `.gitignore`。
- 如果 Key 曾经暴露在聊天记录、截图或公开仓库中，建议在火山引擎控制台轮换密钥。
- 纯前端网页不适合直接保存 AK/SK。本工具使用本地 Python 后端完成签名调用，浏览器只作为操作界面。
